# 1-2-1 登出確認對話框

# Mermaid
```mermaid
sequenceDiagram
  actor User
  participant Browser

  User ->> Browser: 點選【登出】
  Browser ->> Browser: 開啟 dialog#logoutConfirmDialog

  alt 點選【取消】
    User ->> Browser: 點選【取消】
    Browser ->> Browser: close()\n不改變登入狀態
  else 點選【確認】
    User ->> Browser: 點選【確認】
    Browser ->> Browser: close()
    Browser ->> Browser: 觸發登出流程（交由 1-2 處理）
  else 點擊視窗外
    User ->> Browser: 點擊 dialog 外部
    Browser ->> Browser: close()
  else 按下 ESC
    User ->> Browser: 按下 ESC
    Browser ->> Browser: preventDefault + close()
  end
```

## Mermaid 備註
- 此圖僅描述「確認對話框」互動；實際清除 token 與導頁見 `1-2`。
- 程式位置：`services/WebUIServer/app/template/partials/logout_confirm_dialog.html`、`services/WebUIServer/app/static/js/logout.js`。
