# 1-1 登入／註冊

# Mermaid
```mermaid
sequenceDiagram
  actor User
  participant Browser
  participant WebUIServer

  User ->> Browser: 開啟登入/註冊入口
  Browser ->> WebUIServer: GET /auth (或 /auth.html)
  WebUIServer -->> Browser: 200 auth.html + static js/css
  Browser ->> Browser: 載入 sign_login.js\n預設顯示「登入」表單

  alt 切換到註冊表單
    User ->> Browser: 點選【註冊】
    Browser ->> Browser: 切換顯示「註冊」表單
  else 切換回登入表單
    User ->> Browser: 點選【Log in】
    Browser ->> Browser: 切換顯示「登入」表單
  end

  opt 送出登入（詳見 1-1-1）
    note over Browser: 參考 1-1-1 登入：/bff/v1/auth/login → /api/v1/auth/login
  end

  opt 送出註冊（詳見 1-1-2）
    note over Browser: 參考 1-1-2 註冊：/bff/v1/auth/signup → /api/v1/auth/signup
  end
```

## Mermaid 備註
- UI：登入/註冊為同一頁（`auth.html`），由前端 JS 切換表單（參考 `services/WebUIServer/app/static/js/sign_login.js`）。
- 缺少的關鍵資訊：登入/註冊頁內「切換表單」的 DOM 結構與 class 切換細節未在功能描述中完整列出；本圖以「切換顯示」抽象表示（假設）。
