# Mermaid
```mermaid
sequenceDiagram
  actor User
  participant Browser
  participant WebUIServer

  User ->> Browser: 開啟任一功能頁（手機寬度）
  Browser ->> WebUIServer: GET /home (或其他頁)
  WebUIServer -->> Browser: 200 base.html + mobile_nav partial\n+ init_user.js + mobile_nav.js

  Browser ->> Browser: 依使用者 role\n顯示 mobile_nav_user 或 mobile_nav_admin
  Browser ->> Browser: 依 active_page 高亮目前導覽項目

  User ->> Browser: 點選底部導覽項目
  Browser ->> WebUIServer: GET /<target_page>
  WebUIServer -->> Browser: 200 目標功能頁

  opt 桌面寬度（RWD）
    note over Browser: 底部導覽列會隱藏，改用 1-3-1 側邊欄（由 CSS 控制）
  end
```

## Mermaid 備註
- 角色切換：由 `init_user.js` 判斷 `role===admin`，切換顯示 `mobile_nav_admin` 或 `mobile_nav_user`。\n- 高亮：由 `mobile_nav.js`/模板 `active_page` 與 CSS 決定（抽象表示）。\n- 登出：行動版登出按鈕會觸發 1-2/1-2-1。\n+
