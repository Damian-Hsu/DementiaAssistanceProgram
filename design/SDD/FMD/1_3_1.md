# 1-3-1 側邊欄

- **功能編號**：1-3-1
- **功能名稱**：側邊欄
- **功能說明**：桌面版使用側邊欄提供主要頁面導覽，並依目前頁面狀態高亮導覽項目；管理員與一般使用者顯示不同的側邊欄項目。手機寬度下側邊欄會隱藏，改用底部導覽列。
- **程式名稱**：services/WebUIServer/app/template/partials/sidebar.html; services/WebUIServer/app/template/partials/sidebar_admin.html; services/WebUIServer/app/template/base.html; services/WebUIServer/app/static/js/init_user.js; services/WebUIServer/app/static/css/layout.css
- **輸入**：系統、滑鼠；側邊欄介面
- **輸出**：螢幕；側邊欄介面、目標功能介面
- **備註**：登出按鈕會跳出是否登出視窗，流程請參考 1-2/1-2-1。

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 系統載入側邊欄介面 | ❶ 依角色切換側邊欄版本，並依目前頁面狀態高亮對應導覽項目。 | ❶ 顯示側邊欄介面並高亮目前頁面 |
| ❷ 滑鼠點選導覽項目 |  | ❷ 切換至目標功能介面 |
