# 2-4-1 事件列表查詢

- **功能編號**：2-4-1
- **功能名稱**：事件列表查詢
- **功能說明**：使用者依關鍵字與日期範圍查詢事件列表，並可切換排序與分頁。
- **程式名稱**：services/WebUIServer/app/static/js/events.js; services/APIServer/app/router/Events/service.py
- **輸入**：鍵盤、滑鼠；事件檢視介面、events資料表
- **輸出**：螢幕；事件列表

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 鍵盤輸入【關鍵字】、【日期範圍】 |  |  |
| ❷ 滑鼠點選【篩選】 | ❷ 依條件查詢 events 資料表並回傳分頁列表。 | ❷ 顯示事件列表 |

## 資料表

### events

| 欄位 | 中文 | 型態 | 長度 | 空值 | 預設值 | 備註 |
|---|---|---|---:|---|---|---|
| start_time | 事件開始時間 | TIMESTAMP WITH TIME ZONE |  | 是 |  | 日期範圍查詢與排序 |
| summary | 摘要 | TEXT |  | 是 |  | 關鍵字搜尋 |
