# 2-1-1 切換日期

- **功能編號**：2-1-1
- **功能名稱**：切換日期
- **功能說明**：使用者在首頁可透過日期選擇器或左右翻頁切換顯示日期；系統會同步更新日期狀態並重新載入該日日記內容與 Vlog 狀態。
- **程式名稱**：services/WebUIServer/app/template/home.html; services/WebUIServer/app/static/js/home.js; services/WebUIServer/app/static/js/diary.js; services/WebUIServer/app/static/js/vlog.js
- **輸入**：滑鼠；首頁介面
- **輸出**：螢幕；首頁介面、日記介面、Vlog介面

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠點選日期選擇器/上一天/下一天 | ❶ 更新目標日期並重新載入該日的日記與 Vlog 資料。 | ❶ 更新日期顯示並顯示該日內容 |
