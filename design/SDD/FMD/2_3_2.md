# 2-3-2 事件關鍵字搜尋

- **功能編號**：2-3-2
- **功能名稱**：事件關鍵字搜尋
- **功能說明**：使用者輸入關鍵字後，系統以 events 欄位比對關鍵字，過濾出包含相關事件的錄影清單。
- **程式名稱**：services/WebUIServer/app/static/js/recordings.js; services/APIServer/app/router/Recordings/service.py
- **輸入**：鍵盤；影片管理介面、events資料表
- **輸出**：螢幕；影片管理介面

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 鍵盤輸入【關鍵字】 | ❶ 依關鍵字查詢 events 並過濾 recordings 清單。 | ❶ 更新搜尋結果列表 |

## 資料表

### recordings; events

| 欄位 | 中文 | 型態 | 長度 | 空值 | 預設值 | 備註 |
|---|---|---|---:|---|---|---|

| recording_id | 錄影ID | UUID |  | 是 |  | events.recording_id（關聯到 recordings.id） |
| summary | 摘要 | TEXT |  | 是 |  | events.summary |
