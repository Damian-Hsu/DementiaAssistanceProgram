# 1-3-2 行動版底部導覽

- **功能編號**：1-3-2
- **功能名稱**：行動版底部導覽
- **功能說明**：行動版以底部導覽列提供主要功能入口，並依目前頁面狀態高亮導覽項目；管理員與一般使用者顯示不同的底部導覽列。桌面模式下底部導覽列隱藏，改用側邊欄。
- **程式名稱**：services/WebUIServer/app/template/partials/mobile_nav.html; services/WebUIServer/app/template/partials/mobile_nav_admin.html; services/WebUIServer/app/template/base.html; services/WebUIServer/app/static/js/init_user.js; services/WebUIServer/app/static/js/mobile_nav.js; services/WebUIServer/app/static/css/mobile_nav.css; services/WebUIServer/app/static/css/layout.css
- **輸入**：系統、滑鼠；底部導覽介面
- **輸出**：螢幕；底部導覽介面、目標功能介面
- **備註**：登出按鈕會跳出是否登出視窗，流程請參考 1-2/1-2-1。

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 系統載入底部導覽介面 | ❶ 依角色切換底部導覽版本，並依目前頁面狀態高亮對應導覽項目。 | ❶ 顯示底部導覽介面並高亮目前頁面 |
| ❷ 滑鼠點選導覽項目 |  | ❷ 切換至目標功能介面 |
