# 2-5-2 指定查詢日期範圍

- **功能編號**：2-5-2
- **功能名稱**：指定查詢日期範圍
- **功能說明**：使用者在 AI 助手工具面板設定開始/結束日期後，系統將日期範圍一併送入聊天請求，供後端查詢關聯資料時縮小範圍。
- **程式名稱**：services/WebUIServer/app/template/chat.html; services/WebUIServer/app/static/js/chat.js; services/APIServer/app/router/Chat/service.py; services/APIServer/app/router/Chat/DTO.py
- **輸入**：滑鼠、鍵盤；AI助手介面
- **輸出**：系統；查詢條件

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠點選【工具】 |  | ❶ 展開工具面板 |
| ❷ 鍵盤輸入【開始日期】、【結束日期】 | ❷ 送出聊天請求時一併帶入 date_from/date_to。 | ❷ 套用日期範圍查詢條件 |
