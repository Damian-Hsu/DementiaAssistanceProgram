# 2-4-3 編輯事件

- **功能編號**：2-4-3
- **功能名稱**：編輯事件
- **功能說明**：使用者編輯事件內容並儲存後，系統更新 events 資料並刷新事件列表。
- **程式名稱**：services/WebUIServer/app/static/js/events.js; services/APIServer/app/router/Events/service.py; services/APIServer/app/router/Events/DTO.py
- **輸入**：滑鼠、鍵盤；事件詳情視窗、events資料表
- **輸出**：螢幕、資料庫；事件列表、events資料表

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠點選【編輯】 |  | ❶ 跳出編輯視窗 |
| ❷ 鍵盤輸入【行為】、【地點】、【摘要】、【物件】 |  |  |
| ❸ 滑鼠點選【儲存】 | ❸ 送出更新請求並更新 events 資料表。 | ❸ 更新事件資料 |
