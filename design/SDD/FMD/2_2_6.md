# 2-2-6 停止串流

- **功能編號**：2-2-6
- **功能名稱**：停止串流
- **功能說明**：使用者點擊停止串流後，系統先顯示停止確認視窗；確認停止後送出停止請求並停止預覽與更新狀態；取消則不停止。
- **程式名稱**：services/WebUIServer/app/template/partials/stop_stream_confirm_dialog.html; services/WebUIServer/app/static/js/camera.js; services/APIServer/app/router/Camera/service.py; services/StreamingServer/app/main.py
- **輸入**：滑鼠、鍵盤；鏡頭介面、停止確認視窗
- **輸出**：螢幕；停止確認視窗、即時預覽畫面

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠點選【停止串流】<br>&nbsp;&nbsp;① 【取消】<br>&nbsp;&nbsp;② 【確認停止】<br>&nbsp;&nbsp;③ 【點擊視窗外】<br>&nbsp;&nbsp;④ 【ESC】 |  | ❶ 跳出停止確認視窗 |
| ❶-① 滑鼠點選【取消】 |  | ❶-① 關閉停止確認視窗 |
| ❶-② 滑鼠點選【確認停止】 | ❶-② 送出停止請求並停止錄影切片與即時預覽。 | ❶-② 更新串流狀態為未串流並停止預覽 |
| ❶-③ 滑鼠點擊視窗外 |  | ❶-③ 關閉停止確認視窗 |
| ❶-④ 鍵盤按下【ESC】 |  | ❶-④ 關閉停止確認視窗 |
