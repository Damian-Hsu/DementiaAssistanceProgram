# 2-4-2 檢視單一事件內容

- **功能編號**：2-4-2
- **功能名稱**：檢視單一事件內容
- **功能說明**：使用者點擊事件詳情後，系統載入單一事件資料並顯示詳情視窗，必要時提供跳轉至對應影片。
- **程式名稱**：services/WebUIServer/app/static/js/events.js; services/APIServer/app/router/Events/service.py
- **輸入**：滑鼠；事件檢視介面、events資料表
- **輸出**：螢幕；事件詳情視窗

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠點選【詳情】 | ❶ 查詢 events 資料表取得單一事件內容。 | ❶ 跳出事件詳情視窗 |

## 資料表

### events

| 欄位 | 中文 | 型態 | 長度 | 空值 | 預設值 | 備註 |
|---|---|---|---:|---|---|---|
| recording_id | 錄影ID | UUID |  | 是 |  | 用於跳轉對應影片 |
| objects | 物件清單 | ARRAY(VARCHAR) |  | 是 |  | 以標籤方式呈現 |
