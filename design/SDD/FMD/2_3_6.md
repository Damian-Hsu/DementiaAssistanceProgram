# 2-3-6 顯示影片事件列表

- **功能編號**：2-3-6
- **功能名稱**：顯示影片事件列表
- **功能說明**：使用者選取影片後，系統載入該錄影的事件清單並顯示於播放器下方。
- **程式名稱**：services/WebUIServer/app/static/js/recordings.js; services/APIServer/app/router/Recordings/service.py
- **輸入**：系統；影片管理介面、events資料表
- **輸出**：螢幕；事件列表

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 系統載入事件列表 | ❶ 依 recording_id 查詢 events 並排序後渲染。 | ❶ 顯示影片事件列表 |

## 資料表

### events

| 欄位 | 中文 | 型態 | 長度 | 空值 | 預設值 | 備註 |
|---|---|---|---:|---|---|---|
| recording_id | 錄影ID | UUID |  | 是 |  | events.recording_id |
| start_time | 事件開始時間 | TIMESTAMP WITH TIME ZONE |  | 是 |  | events.start_time |
| summary | 摘要 | TEXT |  | 是 |  | events.summary |
