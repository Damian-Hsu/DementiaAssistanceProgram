# 1-1 權限控管

- **功能編號**：1-1
- **功能名稱**：權限控管
- **功能說明**：提供系統的登入與註冊入口，負責切換登入與登出介面。
- **程式名稱**：services/WebUIServer/app/template/auth.html; services/WebUIServer/app/static/js/sign_login.js;
- **輸入**：滑鼠；登入/註冊頁面
- **輸出**：螢幕；登入頁面、註冊頁面

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠停留在登入頁面 |  | ❶ 顯示登入頁面 |
| ❷ 滑鼠點選【註冊】 | ❷ 切換頁面 | ❷ 顯示註冊頁面 |
| ❸ 滑鼠點選【Log in】 | ❸ 切換頁面 | ❸ 顯示登入頁面 |
