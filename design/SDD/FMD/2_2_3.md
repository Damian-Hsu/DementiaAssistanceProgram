# 2-2-3 刪除鏡頭

- **功能編號**：2-2-3
- **功能名稱**：刪除鏡頭
- **功能說明**：使用者刪除鏡頭時，後端採邏輯刪除（將 camera.status 設為 deleted）；前端重新載入鏡頭清單並隱藏已刪除鏡頭，若無可用鏡頭則顯示新增鏡頭表單。
- **程式名稱**：services/WebUIServer/app/static/js/services/CameraService.js; services/WebUIServer/app/static/js/APIClient.js; services/APIServer/app/router/Camera/service.py
- **輸入**：滑鼠；鏡頭介面、camera資料表
- **輸出**：螢幕、資料庫；鏡頭介面、camera資料表

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠選擇【deleted】 | ❶ 送出刪除請求並將 camera.status 設為 deleted。 | ❶ 標記鏡頭為deleted |

## 資料表

### camera

| 欄位 | 中文 | 型態 | 長度 | 空值 | 預設值 | 備註 |
|---|---|---|---:|---|---|---|
| status | 鏡頭狀態 | ENUM |  | 否 | inactive | 刪除時設為 deleted |
| updated_at | 更新時間(UTC) | TIMESTAMP WITH TIME ZONE |  | 是 |  |  |
