# 2-4-4 刪除事件

- **功能編號**：2-4-4
- **功能名稱**：刪除事件
- **功能說明**：使用者刪除事件時需確認；確認後系統刪除 events 資料並刷新列表。
- **程式名稱**：services/WebUIServer/app/static/js/events.js; services/APIServer/app/router/Events/service.py
- **輸入**：滑鼠；事件檢視介面、events資料表
- **輸出**：螢幕、資料庫；事件列表、events資料表

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 滑鼠點選【刪除】<br>&nbsp;&nbsp;① 【取消】<br>&nbsp;&nbsp;② 【確認】 |  | ❶ 跳出刪除確認視窗 |
| ❶-① 滑鼠點選【取消】 |  | ❶-① 關閉刪除確認視窗 |
| ❶-② 滑鼠點選【確認】 | ❶-② 刪除 events 資料並刷新列表。 | ❶-② 刪除事件資料 |

## 資料表

### events

| 欄位 | 中文 | 型態 | 長度 | 空值 | 預設值 | 備註 |
|---|---|---|---:|---|---|---|
| id | 事件ID | UUID |  | 否 | uuid7 | 刪除目標 |
