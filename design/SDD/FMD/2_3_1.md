# 2-3-1 錄影列表查詢

- **功能編號**：2-3-1
- **功能名稱**：錄影列表查詢
- **功能說明**：使用者在影片管理頁載入或調整篩選條件時，系統查詢錄影清單並渲染列表。
- **程式名稱**：services/WebUIServer/app/static/js/recordings.js; services/WebUIServer/app/static/js/APIClient.js; services/APIServer/app/router/Recordings/service.py
- **輸入**：系統、鍵盤、滑鼠；影片管理介面、recordings資料表
- **輸出**：螢幕；影片管理介面

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 系統載入影片管理介面 | ❶ 查詢 recordings 資料表並渲染錄影清單（過濾無效 start_time）。 | ❶ 顯示錄影清單 |

## 資料表

### recordings

| 欄位 | 中文 | 型態 | 長度 | 空值 | 預設值 | 備註 |
|---|---|---|---:|---|---|---|
| start_time | 開始時間 | TIMESTAMP WITH TIME ZONE |  | 是 |  | recordings.start_time（排序/篩選） |
| duration | 影片長度 | FLOAT |  | 是 |  | recordings.duration |
| thumbnail_s3_key | 縮圖S3路徑 | TEXT |  | 是 |  | recordings.thumbnail_s3_key |
