# 1-2-1 登出確認對話框

- **功能編號**：1-2-1
- **功能名稱**：登出確認對話框
- **功能說明**：當使用者點擊登出按鈕時，系統跳出「是否登出」確認視窗；使用者可選擇取消、確認、點擊視窗外部或按 ESC 關閉。若確認則觸發登出流程（清除本機認證資料並導向登入介面）；若取消或關閉則不改變登入狀態。
- **程式名稱**：services/WebUIServer/app/template/base.html; services/WebUIServer/app/template/partials/logout_confirm_dialog.html; services/WebUIServer/app/static/js/logout.js; services/WebUIServer/app/static/css/logout_dialog.css
- **輸入**：滑鼠、鍵盤；是否登出視窗
- **輸出**：螢幕；是否登出視窗、登入介面
- **備註**：此檔僅描述確認視窗行為；實際登出清除 token 與導頁流程請參考 1-2。

## INPUT / PROCESS / OUTPUT

| INPUT | PROCESS | OUTPUT |
|---|---|---|
| ❶ 系統跳出是否登出視窗<br>&nbsp;&nbsp;① 【取消】<br>&nbsp;&nbsp;② 【確認】<br>&nbsp;&nbsp;③ 【點擊視窗外】<br>&nbsp;&nbsp;④ 【ESC】 |  | ❶ 顯示是否登出視窗 |
| ❶-① 滑鼠點選【取消】 |  | ❶-① 關閉是否登出視窗 |
| ❶-② 滑鼠點選【確認】 | ❶-② 觸發登出流程（清除 localStorage.jwt 與使用者快取資訊）。 | ❶-② 切換至登入介面 |
| ❶-③ 滑鼠點擊視窗外 |  | ❶-③ 關閉是否登出視窗 |
| ❶-④ 鍵盤按下【ESC】 |  | ❶-④ 關閉是否登出視窗 |
